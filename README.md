# AI-Chat 智能助手

一个基于React和TypeScript的智能聊天应用，支持多种专业领域的AI助手。

## 功能特性

### 🤖 通用AI助手
- 代码编写和调试
- 概念解释和学习指导
- 问题解决和技术咨询
- 创意写作和内容创作

### 💰 智能理财助手
- **投资分析**
  - 投资回报率计算
  - 风险评估和建议
  - 投资组合优化

- **财务规划**
  - 个人财务目标设定
  - 预算制定和管理
  - 应急基金规划

- **财务计算**
  - 贷款还款计算（房贷、车贷等）
  - 复利收益计算
  - 退休规划计算

- **风险管理**
  - 风险承受能力评估
  - 保险需求分析
  - 资产配置建议

## 技术栈

- **前端**: React 18 + TypeScript
- **UI框架**: Tailwind CSS
- **图标**: Lucide React
- **Markdown渲染**: React Markdown
- **代码高亮**: Rehype Highlight
- **AI集成**: OpenAI API
- **构建工具**: Vite

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置环境变量

创建 `.env` 文件并添加你的OpenAI API密钥：

```env
OPENAI_API_KEY=your_openai_api_key_here
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
```

## 使用指南

### 切换AI助手

应用支持两种AI助手模式：

1. **通用助手** - 处理一般性问题，如编程、学习、创意等
2. **理财助手** - 专门处理财务相关问题

点击左上角的切换按钮可以在两种模式之间切换。

### 理财助手使用示例

#### 投资回报率计算
```
我想投资10万元，年化收益率8%，5年后能赚多少？
```

#### 房贷还款计算
```
帮我计算房贷100万，利率4.5%，30年每月还款多少？
```

#### 退休规划
```
我今年30岁，计划65岁退休，现在有20万存款，每月投入5000元，预期年化收益6%，退休时有多少钱？
```

#### 风险评估
```
我想评估投资风险等级
```

#### 应急基金规划
```
我需要准备多少应急基金？
```

### 通用助手使用示例

#### 编程问题
```
帮我写一个React组件
```

#### 学习指导
```
解释一下什么是机器学习
```

#### 创意写作

帮我写一个故事开头

## 理财工具功能

### 投资回报率计算器
- 支持年复利、月复利、日复利
- 自动计算最终金额、总收益、投资回报率
- 提供投资建议和风险提醒

### 贷款还款计算器
- 支持房贷、车贷等各种贷款类型
- 计算月还款额、总还款额、总利息
- 提供还款策略建议

### 风险评估系统
- 基于年龄、收入、投资目标等因素
- 评估风险承受能力
- 提供个性化投资建议

### 应急基金计算器
- 根据月支出、工作稳定性、家庭规模
- 计算合适的应急基金金额
- 提供财务安全建议

### 退休规划工具
- 基于当前年龄、退休年龄、储蓄情况
- 计算退休时资产总额
- 提供退休规划建议

### 资产配置建议
- 基于年龄和风险承受能力
- 提供股票、债券、现金配置比例
- 包含投资策略说明

## 开发说明

### 添加新的AI助手

1. 创建新的助手组件（如 `src/NewAgent.tsx`）
2. 在 `App.tsx` 中添加新的agent类型
3. 实现相应的切换逻辑和UI

### 扩展理财功能

1. 在 `FinanceAgent.tsx` 的 `financeTools` 对象中添加新工具
2. 在 `processFinanceMessage` 函数中添加相应的处理逻辑
3. 更新欢迎消息中的功能说明

## 部署

### Vercel部署

1. 将代码推送到GitHub
2. 在Vercel中导入项目
3. 配置环境变量 `OPENAI_API_KEY`
4. 部署完成

### 其他平台

项目使用Vite构建，可以部署到任何支持静态网站的平台上。
